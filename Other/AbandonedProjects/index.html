2 Player Games

This game is actually in a somewhat playable state! Though I'll possibly remake this first if I were to go back to revisiting some of my older games. Have you heard of games like Supreme Duelist Stickman, Stickman Party, 2 3 4 Player Games, or Slapstick Fighter? What these games have in common is that unlike online multiplayer games, these games are local multiplayer on one mobile touchscreen. They have very simple controls (Sometimes even only one button to control everything) and usually offer a lot of minigames to play.

So I was inspired to make my own game similar to said games, and I wanted to do it differently so it sticks out from the crowd. I was going to add an online multiplayer system and a map maker (A very simple one.). But as I gotten more and more into the development, I wanted to add more features like 4 players and cosmetics. Howver I couldn't add them because of the way I've programmed them. So for now, I've left this game limbo.
